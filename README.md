# 🚀 SkillGap – Smart Skill Matching Platform

SkillGap is a web application that helps users identify missing skills required for their desired roles and provides smart suggestions to bridge that gap.

---

## 📌 Features

* 🔍 **Skill Gap Analysis**
  Compare your current skills with industry requirements

* 🎯 **Smart Recommendations**
  Suggests skills, tools, and technologies to learn

* 👤 **User Input System**
  Users can enter their current skills and target role

* 🧠 **Dynamic Suggestions (Future Scope)**
  AI-based recommendations (planned)

* 💾 **Database Integration**
  Stores user data using MongoDB

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/skillgap.git
cd skillgap
```

---

### 2. Install dependencies

```bash
npm install
```

---

### 3. Start MongoDB

Make sure MongoDB is running locally:

```
mongodb://127.0.0.1:27017/
```

---

### 4. Run the server

```bash
node server.js
```

---

### 5. Open in browser

```
http://localhost:3000
```
